# 字符串类型

A string is a sequence of bytes or characters, enclosed within either single quote (`'`) or double quote (`"`) characters. Examples:

字符串是是由零个或多个[字符](https://zh.wikipedia.org/wiki/字符)组成的有限序列，包裹在双引号""或者单引号''里面。

```SQL
'a string'
"another string"
```
