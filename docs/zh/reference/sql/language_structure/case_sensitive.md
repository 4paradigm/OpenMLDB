# 大小写敏感

| Category                                             | Case Sensitive? |
| ---------------------------------------------------- | --------------- |
| Keywords                                             | No              |
| Function names                                       | No              |
| Table names                                          | No              |
| Column names                                         | No              |
| All type names except for protocol buffer type names | No              |
| String values                                        | Yes             |
| String comparisons                                   | Yes             |
| `LIKE` matching                                      | Yes             |
